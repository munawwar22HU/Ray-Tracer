Code Features

The following our the features that we implemented in our code.

BRDF

We used the Bi-Directional Reflectance Distribution Function(BRDF's) to compute how is light reflected from opaque surfaces

Cameras

The Cameras we implemented were Parallel and Perspective

Materials

The materials that we implemented where Phong, Cosine, Matte and Reflective Materials

Lights

The lights that we implemented were Spotlight, Point Light, Directional Light and Ambient Light

Acceleration Structures

The acceleration structure that we used to speed up rendering was Bounding Volume Heirarchy

Geometry

We used primitives such as Spheres, Planes and Triangles

Tracers

We implemented a Basic Ray Tracer and a Whitted Ray Tracer

Multi-Threading

We used parallelized our code by using the Library OpenMP.
